有关分支的操作

查看本地和远程所以分支{git branch -a}
删除本地分支{git branch -D 分支名称}
删除远程分支{git push origin -d 远程分支的名称}
创建本地分支并切换到分支{git checkout -b 分支名称}
在当前分支创建分支{git branch 分支名称}（当前为master分支的时候和上面创建的分支一样，）【在创建分支的时候，分支总是以当前的分支内容的起点来进行内容增加减少的修改】
查看修改过的内容状态{git status}
将内容提交到暂存区{git add .}
将内容提交到本地仓库区{git commit -m “这里解释文本”}
将分支提交到远程分支仓库区中去，假如远程分支还没有，则同时创建{git push origin 本地分区名}
